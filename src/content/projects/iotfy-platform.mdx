# IoTfy Platform

The ultimate MQTT-based IoT Dashboard for professionals and hobbyists. Monitor sensors, control devices, and automate your smart home or industrial projects with a beautiful, real-time, and offline-first interface.

![IoTfy Dashboard](/project/iotfy-mobile/iot.png)

## Overview

IoTfy Platform is a cross-platform mobile application built with Flutter that serves as a universal MQTT dashboard. It allows you to connect to any broker (HiveMQ, Mosquitto, EMQX, AWS IoT, Azure) and manage multiple connections simultaneously with enterprise-grade MQTTS security support.

## Tech Stack

- **Frontend:** Flutter, Dart
- **State Management:** Riverpod
- **Local Database:** Hive (NoSQL)
- **Messaging:** mqtt_client
- **Visualization:** fl_chart, Syncfusion Gauges

## Features

- **Universal MQTT Support:** Connect to any broker with SSL/TLS secure connection support.
- **Rich Visualization:** Dynamic widgets including sliders, knobs (PWM/Dimmer), toggle switches, real-time charts, gauges, and text displays.
- **Intelligent Automation (Rule Engine):** Built-in offline-first automation engine with widget value and schedule-based triggers.
- **Premium User Experience:** Fully optimized Dark/Light mode, zero latency real-time updates, and highly responsive design for both touch and mouse interactions.
- **Multi-Broker Management:** Manage and monitor multiple connections and dashboards simultaneously.
