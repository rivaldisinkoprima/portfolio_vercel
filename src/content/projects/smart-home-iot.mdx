# Smart Home IoT Hub

A comprehensive IoT solution for smart home automation featuring real-time device control and monitoring.

## Overview

This project implements a centralized hub for managing various smart home devices including lights, temperature sensors, and security cameras.

## Tech Stack

- **Hardware:** ESP32, Raspberry Pi
- **Backend:** Node.js, MQTT Broker
- **Frontend:** React, Tailwind CSS
- **Database:** InfluxDB for time-series data

## Features

- Real-time device monitoring
- Automated scheduling
- Energy consumption tracking
- Mobile-responsive dashboard
