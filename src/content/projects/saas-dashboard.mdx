# SaaS Analytics Dashboard

A modern analytics dashboard for SaaS companies to track key performance indicators and user behavior.

## Overview

Built with performance in mind, this dashboard provides real-time insights into business metrics.

## Tech Stack

- **Frontend:** Next.js 15, TypeScript, Tailwind CSS
- **Backend:** Next.js API Routes, Prisma
- **Database:** PostgreSQL
- **Charts:** Recharts

## Features

- Real-time data visualization
- Custom report generation
- User segmentation
- API for data export
